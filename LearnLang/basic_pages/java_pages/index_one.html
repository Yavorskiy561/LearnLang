Переменные

Концепция переменных и типов данных является основополагающей в языке программирования C. Переменная - это именованное место в памяти, которое может хранить значение определенного типа данных. Например:

int x = 5

В приведенном примере мы объявляем переменную с именем "x", в переменной "x" присваиваем ей значение 5.

Типы данных

Типы данных определяют вид данных, которые может содержать переменная. В C существует несколько основных типов данных, которые можно использовать для объявления переменных. Рассмотрим каждый тип данных подробнее.

Целочисленные типы данных

int: используется для хранения целых чисел. Занимает обычно 4 байта.
Пример:

int age =25;

short: используется для хранения коротких целых чисел. Занимает обычно 2 байта.
Пример:

short temperature = -10;

long: используется для хранения длинных целых чисел. Занимает обычно 8 байт.
Пример:

long population = 7894561230;

Вещественные типы данных

float: используется для хранения чисел с плавающей точкой одинарной точности. Занимает обычно 4 байта.
Пример:

float pi = 3.14159;

double: используется для хранения чисел с плавающей точкой двойной точности. Занимает обычно 8 байт.
Пример:

long population = 7894561230;

Символьный тип данных

char: используется для хранения символов. Занимает 2 байта и представляет символ в кодировке Unicode.
Пример:

char grade = 'A';

Строки

String: используется для хранения текстовых строк.
Пример

String grade = 'John';

Логический тип данных:

bool: в C нет встроенного типа bool, но его можно эмулировать с помощью int или char. Обычно используется 0 для false и любое ненулевое значение для true.
Пример:

bool isTrue = true;
bool isTrue = false;

Константы

final: используется для объявления переменных, значения которых не могут быть изменены.
Пример:

final float pi = 3.14159;

Массивы

Тип [количество символов]: массив - это набор данных одного типа
Пример:

int[] name = {1, 2, 3, 4, 5};

Объектные типы данных

Классы: в Java также можно создавать пользовательские типы данных с помощью классов. Классы позволяют определить состояние и поведение объектов.
Пример:

class Person {
String name;
int age;
}
public static void main(String[] args) {
Person person = new Person();
person.name = "John";
person. age25;
}

Классы-оболочки: Java также предоставляет классы-оболочки для каждого примитивного типа данных. Например, Integer для типа int, Double для типа double и т.д. Классы-оболочки позволяют работать с примитивными типами данных как с объектами.
Пример:

Integer number = 10;
Double pi = 3.14159;
